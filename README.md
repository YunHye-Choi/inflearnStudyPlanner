# How to use
- 프로그램 실행 후 표준 입력에 김영한 강사님 인프런 커리큘럼 페이지를 섹션 1 부분부터 드래그하여 복사, 붙여넣기 한다.
- 표준 출력으로 'DAY N \t 시작 범위 \t 끝 범위 \t 총 시간' 형태로 출력된다
- 해당 데이터를 엑셀에 붙여넣으면 28분에 가까운 일정량을 공부할 스터디 플래너가 완성된다.
